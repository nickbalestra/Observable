# Stream

A toy functional reactive stream library for JavaScript (Similar to RxJS, XStream, MostJS, ...)

## Factories:

- Stream.of()
- Stream.fromArray()
- Stream.fromPromise()
- Stream.fromEvent()
- Stream.interval()

## Composing Operators:

- Stream.map()
- Stream.mapTo()
- Stream.filter()
- Stream.takeUntil()
